# Bittee Audio Assets

Audio assets repository for the Bittee Falasteeni game.

This repository hosts audio files (WebM format) that are used by the game. The files are served via GitHub Pages for compatibility with hosting platforms that restrict audio file uploads.

## Repository Setup Instructions

1. **Repository Name**: `bittee` (as you mentioned)
2. **Owner**: `bittee-falasteeni`
3. **Visibility**: **Public** (required for GitHub Pages free tier)
4. **Initialize with README**: ✅ Yes
5. **Add License**: ✅ Yes (MIT or similar open license recommended)

## Description for Repository

```
Audio assets for Bittee Falasteeni - A web-based game. Hosted via GitHub Pages for cross-platform compatibility.
```

## After Creating Repository

1. Clone the repository
2. Copy all `.webm` files from `public/assets/audio/` to the repository
3. Enable GitHub Pages in repository settings
4. Update game code to load audio from: `https://bittee-falasteeni.github.io/bittee/`
